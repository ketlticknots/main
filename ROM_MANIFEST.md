# ROM Manifest

This directory contains curated ROM files for the EmulatorJS retro gaming platform.

## Available Emulators

- NES (Nintendo Entertainment System)
- SNES (Super Nintendo Entertainment System)
- Genesis (Sega Genesis)
- Game Boy / Game Boy Color
- N64 (Nintendo 64)
- PSX (PlayStation 1)
- Arcade (MAME/FBA)
- Atari 2600
- TurboGrafx-16
- Neo Geo

## How to Add ROMs

1. Place ROM files in the appropriate system folder
2. Ensure files are in the correct format (.nes, .smc, .gen, .gb, etc.)
3. ROM files must be licensed or public domain
4. Update this manifest with new additions
