## Troubleshooting

**Game won't load:**
- Verify ROM format matches system selection
- Try a different ROM file
- Clear browser cache and reload

**Slow performance:**
- Check FPS counter
- Close other browser tabs
- Try a lower-end game (NES vs N64)

**Saves disappeared:**
- Browser cache was cleared
- Switching to private/incognito mode
- Use "Save State" feature for backup

**Controls not working:**
- Click emulator screen first to focus
- Try keyboard if touch controls fail
- Check keyboard layout (QWERTY vs AZERTY)

## File Upload Security

- ROMs are processed locally in your browser
- No files are uploaded to servers
- Games stored in browser's localStorage only
- Safe to upload any legal ROM file
